Ⅰ. Breviarium

If you don't know why you're here: this document describes some mechanical bits in the online game Arcanum. This is not intended as a general-purpose guide.

This is a document generated by pulling data from the Arcanum repository, from the master branch containing Wizrobe (Theory Of Magic) version 0.10.15. A Python script processes these data, and then fills out a Jinja template.

Since this script operates somewhat "close to the metal", it doesn't put a whole lot of effort into translating game code elements into legible form. For example, any mention of evt_helper is left as-is and there are no hints (other than here) that that's actually an internal flag tracking the status of "job"-tier classes.
